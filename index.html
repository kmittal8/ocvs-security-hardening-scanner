VMware vSphere Security Hardening Scanner

An automated SOC compliance scanning and remediation platform for enterprise vSphere environments, deployed on Oracle Cloud Infrastructure.

🔍 What This Project Does

This project is an enterprise-grade VMware vSphere Security Compliance Scanner built to automate the tedious manual work of hardening vSphere 8 environments. It connects directly to a vCenter instance, runs 60+ compliance checks against the VMware vSphere Security Configuration Guide v8 (SCG v8), and produces audit-ready results with AI-generated narratives and PDF reports.

The application is packaged as a Streamlit web dashboard and deployed to a dedicated OCI compute instance via Terraform, making it accessible from a browser without any local tooling beyond SSH credentials.

🚀 Deployment Overview

The scanner runs on an OCI compute instance in the ap-melbourne-1 region (public IP 161.33.92.184, Streamlit port 8501). To deploy or redeploy from scratch:

Quick-start — from terraform/ocvs_scanner_vm/
# 1. Copy the example config and fill in your OCI credentials
cp terraform.tfvars.example terraform.tfvars

# 2. Initialise Terraform providers
terraform init

# 3. Preview the resources that will be created
terraform plan

# 4. Deploy the OCI VM + security list
terraform apply

# 5. SSH into the instance (key is auto-generated by Terraform)
ssh -i id_rsa_ocvs_scanner opc@<public-ip>

# 6. Start the Streamlit app (run on the VM)
streamlit run app.py --server.port 8501

# 7. Open in browser
http://<public-ip>:8501
